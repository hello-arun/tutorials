To convert values from GPa to N/m for 2D materials multiply the GPa values to lattice param(in Nano Meter) along out of plane direction.

So here for example

C11 = 10.7047 GPa

Since 20 Angstrom = 2 Nanometer

C11 = 21.4 N/m
